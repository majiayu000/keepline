# 项目上下文

## 项目概述

开发一个 Timberman（伐木人）风格的 HTML5 小游戏。

**游戏规则：**
1. 玩家控制伐木工，可以站在树的左边或右边
2. 按左/右键砍树，伐木工会移动到对应方向并砍一刀
3. 树干会不断下落，树枝随机出现在左边或右边
4. 如果伐木工碰到树枝，游戏结束
5. 有时间条，砍树会增加时间，不砍树时间会减少
6. 时间耗尽也会游戏结束
7. 目标是砍尽可能多的树获得高分

**技术栈：**
- HTML5 Canvas
- 纯 JavaScript（无框架）
- CSS3 动画

**文件结构：**
```
workspace/
├── index.html      # 主页面
├── style.css       # 样式
├── game.js         # 游戏逻辑
└── assets/         # 资源（如果需要）
```
